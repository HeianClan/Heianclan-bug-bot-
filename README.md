1. Download and install Termux from F-Droid.
2. Move this folder to Termux storage.
3. Run:
```
cd EvilHowlV10
chmod +x start.sh
./start.sh
```

4. Scan the QR code from WhatsApp.
5. Type !ping or !menu to test.
